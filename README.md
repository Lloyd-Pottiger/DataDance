## Development

TTY 1:

```shell
cd backend
yarn   # Install dependencies
yarn start
```

TTY 2:

```shell
cd ui
yarn
yarn gen:client  # Generate API client from the backend
yarn start
```

Then, nagivate the browser to http://localhost:3000/.
